# A simple TODO app with user authentication

## Setup and running instructions

Requirements:
- Node Version Manager (nvm) and Node v. 14
- Yarn
- Docker

### Install Docker

### Install NVM

### Install and use Node

```bash
$ nvm install 14
$ nvm use
```

### Install Yarn

```bash
$ npm i -g yarn
```

### Install dependencies

```bash
$ yarn
```
